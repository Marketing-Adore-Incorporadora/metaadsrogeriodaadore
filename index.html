Como atualizar o dashboard online
1. Clique em "Exportar JSON" → baixa o data.json atualizado.
2. Acesse github.com/Marketing-Adore-Incorporadora/metaadsRogeriodaAdore
3. Clique no arquivo data.json → lápis → selecione tudo → cole o novo conteúdo → "Commit changes".
4. Em 1–2 minutos o site atualiza para todos.
