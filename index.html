Como atualizar o dashboard online (GitHub Pages)
1. Clique em "Exportar JSON" acima para baixar o arquivo data.json atualizado com todos os períodos.
2. Acesse seu repositório no GitHub (github.com/seuusuario/meta-ads-dashboard).
3. Clique no arquivo data.json → clique no lápis (editar) → cole o conteúdo novo → "Commit changes".
4. Pronto — em 1–2 minutos o site atualiza e qualquer pessoa com o link vê os dados novos.
